# Project Modules

## 1. Product Module

Handles product creation, update, listing, and search.

## 2. Warehouse Module

Stores warehouse details like city, pincode, address, latitude, longitude, and service radius.

## 3. Inventory Module

Tracks stock per product per warehouse.

## 4. Customer Location Module

Handles delivery pincode, city, latitude, and longitude.

## 5. Order Module

Handles order placement, order items, order status, and allocated warehouse.

## 6. Warehouse Allocation Engine

Decides which warehouse should fulfill the order.

## 7. Notification Module

Creates low-stock notifications.

## 8. Dashboard Module

Provides analytics for admin dashboard.

## 9. React Frontend

Displays products, warehouses, inventory, orders, and alerts.

## 10. Docker Setup

Runs backend, PostgreSQL, and frontend using containers.
